ASL  TO TEXT

In a time when isolation and social distancing is a norm, there is a greater need for better communication and interaction amongst the people. Modern developments are working largely towards this goal of connecting people. A section of society that needs technology to help them communicate more than anyone else is the deaf and dumb. Many new technologies and researches have emerged to help this particular section in communicating better. We create a model that can convert sign language to text in real time. The model takes in a live feed of the image, and with the help of a trained convolution neural network model with ASL hand sign images for their classification, we predict the hand sign

>

HOW IT WORKS?

The project takes in a dataset containing several images for each alphabet character. We train the model using CNN, and various Keras Libraries. The model is then ready to work to recognize real time signs and convert them to text. Previously people have tried to make a model which involves a hardware device to collect the gestures such as gloves. This has resulted in a limit to the scope of the project. As the number of people who can use the model become limited, also the number of actions that can be taken also become limited by it. A specific background and controlled environmental setting are also a common feature of various models. What we have tried to achieve here is to create a real time model without these boundaries.
